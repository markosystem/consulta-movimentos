# Consulta de Movimentos

## Prerequisites
NPM 6.13 and Java 8

## Start application front-end
Run `npm start` for a dev server. Navigate to `http://localhost:4200/`.

## Start application back-end api
Run `./mvnw springboot:run` or user Intellij IDE/Eclipse with maven. Navigate to `http://localhost:8080/swagger-ui.html` to perform tests on the api.
